# Санкт-Петербургский парадокс

## Формулировка
Некоторое казино предлагает следующую игру:
- вступая в неё, игрок платит некоторую фиксированную сумму
- на каждом шаге игрок подбрасывает монету
- если выпадает "орёл", то игра заканчивается и игрок получает выигрыш, иначе игра продолжается
- начальная сумма выигрыша равна 2
- на каждом последующем шаге сумма выигрыша возрастает в 2 раза

Вопрос - какая вступительная сумма является справедливой?

В чистой математической формулировке игра может быть бесконечной, а значит и сумма выигрыша может быть равна бесконечности. Однако, мало кто из
здравомыслящих людей готов заплатить за право сыграть в эту игру даже небольшую сумму.

Существуют различные идеи разрешения этого парадокса, часть из них вводят дополнительные ограничения, которые бы присутствовали в реальной жизни.

Мы рассмотрим одно такое ограничение: казино не обладает бесконечными ресурсами и таким образом сумма выигрыша ограничена сверху. Можно оценить матожидание
выигрыша в такой постановке и предложить это значение в качестве справедливой платы за участие.

## Задача
Имея заготовку программы, напишите функцию, вычисляющую оценку матожидания выигрыша, принимающую аргументы:
- количество итераций моделирования
- максимальная сумма, которую казино может выплатить одному игроку

Функция должна возвращать одно вещественное число - значение матожидания.

Заготовка реализации находится в файле `src/st_petersburg.cpp`, для получения очередного случайного числа нужно воспользоваться функцией `get_random_number()`.
